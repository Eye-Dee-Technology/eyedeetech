Senior Full-Stack Developer
Remote | Full-time • Senior

We're seeking an experienced engineer to own features end-to-end — from UI to backend services and cloud infrastructure.

Responsibilities
  • Design and implement scalable web applications using React (or similar) and Node.js.
  • Own API design, data models, and application performance tuning.
  • Collaborate with product and design to deliver high-quality user experiences.
  • Mentor junior engineers and participate in code reviews and architecture discussions.
Requirements
  • 5+ years building production web applications.
  • Strong experience with React, modern JS, Node.js/Express, and REST/GraphQL APIs.
  • Familiarity with cloud platforms (AWS/GCP/Azure), IaC (Terraform/CloudFormation), and CI/CD.
  • Good testing practices (unit, integration, E2E) and observability (logs, metrics).
Nice to have
  • Experience with TypeScript, microservices, and container orchestration (Kubernetes).
  • Previous work on regulated or security-sensitive systems.
Benefits & perks
  • Competitive salary and equity options.
  • Flexible hours and remote-first culture.
  • Learning budget and conference allowance.

How to apply: Email a short note, your resume, and a link to your portfolio or GitHub to careers@eyedeetech.com with subject Senior Full-Stack Developer — Application.

Security Engineer
Remote | Full-time • Mid/Senior

  • Perform threat modeling and security reviews of designs and code.
  • Develop and maintain security tooling (SAST/DAST integrations, CI checks).
  • Lead incident response, vulnerability triage, and remediation planning.
  • Run or coordinate penetration tests and collaborate with third-party auditors.

  • 3+ years in security engineering, application security, or related roles.
